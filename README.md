# Front-End Sistema de Gerenciamento de Despesas

## Configuração do projeto

- Instalar as Dependências
```
npm install
```

- Iniciar com o comando
```
npm run serve
```

## Tecnologias
- Vue Hooks Form
- Vue Toastification
- Axios
- oh-vue-icons
- vue-toastification

## O que falta ser feito ?
- Filtros ( categorias, valor, data )
- Paginação
